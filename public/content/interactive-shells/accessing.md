# Accessing Interactive Shells

| Language | Input      | Exit   |
| -------- | ---------- | ------ |
| ruby     | irb        | exit   |
| php      | php -a     | exit   |
| js       | node -i    | .exit  |
| python   | python -i  | exit() |
| python3  | python3 -i | exit() |
| go       | gore       | ^D     |
