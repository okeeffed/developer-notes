## Creating Instances

```
>>> class Monster:
...     hit_points=1
...     color="blue"
...     weapon="sword"
...
>>> monster = Monster()
>>> monster.hit_points
1
>>> monster.weapon
'sword'
>>> monster.color
'blue'

# jubjub instance
>>> jubjub = Monster()
>>> type(jubjub)
<type 'instance'>
>>> jubjub.hit_points
1
>>> jubjub.hit_points = 5
>>> jubjub.hit_points
5
```

		self.hit_points = kwargs.get('hit_points', 5)
		self.weapon = kwargs.get('weapon', 'sword')
		self.color = kwargs.get('color', 'yellow')
		self.sound = kwargs.get('sound', 'yell')

monster = Monster(hit_points=22, color="green")

class Character:
