# Getting started with Helm

## Installation

```s
brew install helm
```

Once Helm is installed, you can check the [Artifact Hub](https://artifacthub.io/packages/search?kind=0) for packages.

Example:

```s
$ helm repo add bitnami https://charts.bitnami.com/bitnami
$ helm search repo bitnami
```
